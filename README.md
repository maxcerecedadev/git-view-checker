# Git View Checker

Una herramienta moderna y rápida para comparar bloques de código y visualizar las diferencias, similar a la vista de diferencias de GitHub.

![DiffChecker Demo](https://via.placeholder.com/800x400?text=DiffChecker+Preview)

## 🚀 Características

- **Comparación en Tiempo Real**: Pega tu código original y modificado para ver las diferencias al instante.
- **Vistas Flexibles**: Alterna fácilmente entre vista **Unificada** y **Dividida** (Split).
- **Interfaz Moderna**: Diseño oscuro, limpio y amigable para desarrolladores.
- **Resaltado de Sintaxis**: Diferenciación clara de líneas añadidas (verde) y eliminadas (rojo).
- **Responsive**: Funciona perfectamente en escritorio y dispositivos móviles.

## 🛠️ Tecnologías Utilizadas

- **React**: Biblioteca principal de UI.
- **Vite**: Entorno de desarrollo ultrarrápido.
- **Diff**: Motor para calcular las diferencias entre textos.
- **Lucide React**: Iconos modernos y ligeros.
- **CSS Modules / Variables**: Estilizado personalizado con tema oscuro.

## 📦 Instalación y Uso Local

1.  **Clonar el repositorio**:

    ```bash
    git clone https://github.com/maxcerecedadev/git-view-checker.git
    cd git-view-checker
    ```

2.  **Instalar dependencias**:

    ```bash
    npm install
    ```

3.  **Iniciar el servidor de desarrollo**:

    ```bash
    npm run dev
    ```

4.  Abre tu navegador en `http://localhost:5173`.
